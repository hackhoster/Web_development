Css Weight & Height

    : CSS Width & Height are commonly specified according their name.

  • Width Css properties sets the width of element.
  • Height Css properties sets the Height of element.
  • max-height
  • min-height
  • max-width
  • min-width


Width

  • Width Css properties sets the width of element.
  • Width haven't auto height
    • div   {

Min Width

  • The min-width property defines the minimum width of an element.
  • If the content is smaller than the minimum width, the minimum width will be applied
  • This prevents the value of the width property from becoming smaller than min-width.
  • Mostly used to make responsive web design.
    • div   {
           min-width: 100px;
    }




    See Result


Max Width

  • The max-width property defines the maximum width of an element.
  • If the content is larger than the maximum width, it will automatically change the height of the element.
  • This prevents the value of the width property from becoming smaller than min-width.
    The value of the max-width property overrides the width property.
  • If the content is smaller than the maximum width, the max-width property has no effect.
  • Mostly used to make responsive web design.
    • div   {
           max-width: 200px;
    }




    See Result


Max Height

  • The max-height property defines the maximum height of an element.
  • If the content is larger than the maximum height, it will overflow. How the container will handle the overflowing content is defined by the overflow property.
  • If the content is smaller than the maximum height, the max-height property has no effect.
  • This prevents the value of the height property from becoming larger than max-height. The value of the max-height property overrides the height property.
  • Mostly used to make responsive web design.
    • div   {
           max-height: 200px;
    }




    See Result


Min Height

  • The min-height property defines the minimum height of an element.
  • If the content is smaller than the minimum height, the minimum height will be applied.
  • If the content is larger than the minimum height, the min-height property has no effect.
  • This prevents the value of the height property from becoming smaller than min-height.
  • Mostly used to make responsive web design.
    • div   {
           min-height: 200px;
    }
